###########运行  
模型的运行需要在树莓派的已经配置好的虚拟环境中。  
首先进入虚拟环境  
然后输入python mycar/manage.py drive --model /home/pi/mycar/models/mypilot9.h5即可运行  
最后打开http://192.168.43.29:8887/并选择对应的模式就可以实现自动驾驶了
